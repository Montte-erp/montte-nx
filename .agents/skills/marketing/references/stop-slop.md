# Stop-slop pass para copy Montte

Use este arquivo como revisão final de blog, LinkedIn e X. Ele adapta a skill [hardikpandya/stop-slop](https://github.com/hardikpandya/stop-slop) para pt-BR e para a voz do Montte.

Objetivo: tirar o cheiro de texto gerado por IA sem deixar a copy informal demais, agressiva demais ou vaga.

## Regra central

A frase precisa ter uma pessoa fazendo algo concreto.

Bom:

> Reescrevemos a tabela porque o fechamento de sexta travava em 800 lançamentos.

Ruim:

> A tabela foi otimizada para oferecer uma experiência mais robusta.

Se a frase não tem ator, número, nome, data ou cena, ela provavelmente precisa de corte.

## Corte abertura muleta

Apague aberturas que anunciam o ponto antes de dizer o ponto.

| Corte | Escreva direto |
|-------|----------------|
| "Aqui está o ponto" | Diga o ponto |
| "A verdade é que" | Diga a verdade |
| "O ponto é que" | Nomeie o problema |
| "Vale notar que" | Mantenha só o fato |
| "É importante lembrar" | Mantenha só o fato |
| "Pense nisso" | Confie no leitor |
| "Isso importa porque" | Mostre o impacto |
| "Neste post, vamos" | Comece pelo que mudou |

Exemplo:

Antes:

> Vale notar que a conciliação manual consome muito tempo dos founders.

Depois:

> Founders perdem sexta à tarde batendo banco, fatura e planilha.

## Evite contraste mecânico

O padrão "não é X, é Y" vira muleta rápido. Use só quando a negativa for posicionamento canônico do Montte.

Permitido por posicionamento:

> Não é Bling, Omie ou Conta Azul.

Corte em quase todo resto:

| Padrão fraco | Melhor |
|--------------|--------|
| "O problema não é cobrança. É visibilidade." | "O founder não sabe quem está devendo até abrir três abas." |
| "Não estamos criando um ERP, estamos criando uma camada de billing." | "Montte é camada de billing pra SaaS." |
| "Não é apenas automação, é clareza financeira." | "O painel mostra inadimplência, uso e fatura numa tela." |
| "A pergunta não é X. A pergunta é Y." | Faça a pergunta certa uma vez ou afirme a tese. |

Se a frase começa negando uma tese que ninguém trouxe, apague a negativa.

## Dê ator humano para a frase

IA esconde responsabilidade com passiva e agência falsa. Montte não.

| Fraco | Forte |
|-------|-------|
| "A tabela foi reescrita" | "Reescrevemos a tabela" |
| "Bugs foram corrigidos" | "Corrigimos 4 bugs" |
| "A decisão surgiu depois dos testes" | "Decidimos isso depois de 6 testes" |
| "Os dados mostram" | "Vimos no PostHog" |
| "A demanda cresceu" | "12 founders pediram isso na waitlist" |
| "A cultura muda" | "O time para de exportar planilha" |

Regra: se um objeto parece estar pensando, decidindo, mostrando ou mudando sozinho, reescreva.

## Troque abstração por cena

Copy abstrata parece segura, mas ninguém lembra.

| Vago | Concreto |
|------|----------|
| "melhor experiência financeira" | "menos abas abertas no fechamento" |
| "mais visibilidade" | "inadimplência aparece antes da reunião de segunda" |
| "processo eficiente" | "3 cliques a menos para emitir fatura" |
| "ganho operacional" | "sexta sem exportar CSV" |
| "decisão estratégica" | "cortar plano que dá prejuízo" |

Use número, nome próprio, data, ferramenta ou cena de trabalho.

## Corte advérbios e amaciadores

Apague palavras que tentam dar força sem adicionar fato.

- realmente
- literalmente
- simplesmente
- basicamente
- muito
- bastante
- extremamente
- genuinamente
- fundamentalmente
- provavelmente, quando usado para fugir de afirmar
- talvez, quando usado por medo de afirmar

Antes:

> O Montte realmente ajuda founders a entenderem melhor o financeiro.

Depois:

> O Montte mostra uso, fatura e status de pagamento por cliente.

## Varie ritmo sem dramatizar

Use frases curtas, médias e longas. Não transforme tudo em punchline.

Evite:

> Cobrança quebrada. Planilha aberta. Founder cansado. Esse é o problema.

Melhor:

> O founder abre a planilha porque o gateway só conta metade da história. Depois cruza fatura, uso e status de pagamento na mão. Cansa.

Duas regras simples:

- Se três frases seguidas têm tamanho parecido, quebre uma ou junte duas.
- Se todo parágrafo termina com frase de impacto, um deles precisa terminar normal.

## Corte frases de palestra

Frase boa demais para quote costuma soar fabricada.

Corte ou reescreva:

- "Ponto final."
- "Deixe isso afundar."
- "E isso muda tudo."
- "Essa é a diferença entre X e Y."
- "No fim, tudo se resume a..."
- "O futuro pertence a..."
- "Isso é o que liderança de verdade parece."

Montte escreve como quem está construindo, não como keynote.

## Checklist de 2 minutos

Antes de publicar, responda:

- [ ] Tem ator humano nas frases principais?
- [ ] Cortei "aqui está", "a verdade é", "o ponto é", "isso importa porque"?
- [ ] Cortei contraste mecânico "não X, mas Y" quando dava para afirmar Y direto?
- [ ] Troquei passiva por ativa?
- [ ] Removi inanimados agindo como pessoas?
- [ ] Cada seção tem número, nome, data, ferramenta ou cena concreta?
- [ ] Cortei advérbios e amaciadores sem fato?
- [ ] O ritmo não ficou metronômico nem teatral?
- [ ] A última frase não parece encerramento de palestra?

## Pontuação

Dê nota de 1 a 10 em cada dimensão:

| Dimensão | Pergunta |
|----------|----------|
| Direto | O texto afirma ou anuncia que vai afirmar? |
| Ritmo | As frases variam sem virar teatro? |
| Confiança | O texto respeita o leitor ou explica demais? |
| Autenticidade | Parece alguém do Montte contando o que aconteceu? |
| Densidade | Tem palavra que dá para cortar sem perder sentido? |

Abaixo de 35/50: reescreva antes de enviar.
